## 🚀 使用方法

### 在 RikkaHub 中使用

1. 開啟 RikkaHub
2. 進入「代理技能」頁面
3. 技能會自動載入

### 手動安裝

```bash
# 克隆倉庫
git clone https://github.com/ook826092-cloud/skills.git

# 將技能目錄複製到 RikkaHub 的 skills 目錄
cp -r skills/* /path/to/rikkahub/skills/
```
