## 🚀 Usage

### Use in RikkaHub

1. Open RikkaHub
2. Go to "Agent Skills" page
3. Skills will load automatically

### Manual Installation

```bash
# Clone repository
git clone https://github.com/ook826092-cloud/skills.git

# Copy skills to RikkaHub skills directory
cp -r skills/* /path/to/rikkahub/skills/
```
