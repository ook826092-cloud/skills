## 他蒸馏了什么

户晨风不是学者，是用个人经历和具体数字讲道理的内容创作者。他的核心认知框架：

| 心智模型 | 一句话 |
|---------|--------|
| **消费选择即生活标准** | 你买什么品牌，就过什么质量的生活——大品牌是经过市场验证的最优解 |
| **苹果人与安卓人** | 消费选择暴露了你的生活标准、认知水平和对自己的要求 |
| **城市筛选标准** | 有山姆、苹果店、国际机场和地铁的城市才值得年轻人待 |
| **购买力即真相** | 别听宣传，用钱量——物价和工资比GDP更能说明真相 |
| **学历现实主义** | 学历在市场上就是硬通货，不满意就去改变，别自欺欺人 |

10条决策启发式，包括：
- 消费别省，省出来的是麻烦
- 判断一个地方值不值得：看它有什么店
- 吃苦不值得被美化
- 收入是最好的话语权
- 铁饭碗不是有人给你饭吃，而是到哪都有饭吃
- 挣钱是最伟大的事

99条一手语录库，全部从直播文字稿直接提取并标注日期。

## 诚实边界

**这个Skill能做的：**
- 用户晨风的消费现实主义框架分析消费、城市、职业选择问题
- 模拟他的直接、数据驱动、绝对化表达风格
- 如实呈现他的内在矛盾和争议

**做不到的：**

| 维度 | 说明 |
|------|------|
| 政治分析 | 他本人极度回避政治话题 |
| 替代本人 | 真实想法、封号后心态和去向无法复制 |
| 消除争议 | "苹果人/安卓人"体系被官方定性为"煽动群体对立"，Skill如实呈现，不做辩护 |
| 早期内容还原 | 2023年前个人经历存在大量空白 |

**增强能力（可选）：**
- 配置向量检索MCP后，可从490份直播文字稿中实时语义搜索相关原始语录

**特别说明**：户晨风是一个**高度争议性**的人物。他的部分观点（如"苹果人/安卓人"分类、对低收入群体的标签化）已被官方媒体定性为"煽动群体对立"。本Skill旨在还原他的思维方式，**不代表认同其所有观点**。

**一个不告诉你局限在哪的Skill，不值得信任。**

## 关于户晨风

户晨风（1998-），江苏镇江人，自媒体创作者。高中学历，做过汽修工，后全职做短视频和直播。以"购买力挑战"系列视频出圈，用具体金额展示不同群体/地区的真实生活状况。后期发展出"苹果人/安卓人"消费分层理论，引发巨大争议。2025年9月全网封禁，12月被中央网信办通报。

他是一面镜子——**你从他身上看到的，取决于你站在哪里。**
