## 📋 任务分配速查

| 难度 | 模型类型 | 适合任务 |
|:---:|:---:|----------|
| ⭐ | 轻量级/快速模型 | 数据整理、信息对比、基础分析 |
| ⭐⭐ | 中等能力模型 | 战术分析、需求分析、对手分析 |
| ⭐⭐⭐ | 强能力模型 | 概率推演、场景分析、风险评估 |
| ⭐⭐⭐⭐ | 最强模型 | 综合判断、最终预测、决策建议 |

## ⚠️ 必须记住

1. **一次性调用** - 不要逐个调用，会超时
2. **主Agent整合** - 不要只是罗列结果
3. **控制token** - 每个模型300-500足够
4. **给出结论** - 最终判断是你的
5. **模型可替换** - 根据实际配置调整
